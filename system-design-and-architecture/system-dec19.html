System Design and Architecture

Section 01: System Design 101

  • Networking - OSI & TCP/IP
    • Open System Interconnection (OSI)

      (ANY kind of information between two systems)

      • 7 Layers
        • 7 - Application Layer (HTTP(S)
          • Holds things closest to the user
        • 6 - Presentation Layer/Translation Layer
          • Data we get from the application layer is extracted and gets manipulated/formatted to be translated across the network
          • Encryption/Decryption, Compression, Translation
        • 5 - Session Layer
          • Responsible for opening/closing the communication between two devices
          • Certain period of time in which a session is alive
          • Responsible for making session is open long enough but still closes to ensure resources not wasted
        • 4 - Transport Layer
          • Responsible for end-to-end communication between two devices
          • Takes data from session layer, splits into sections or chunks
        • 3 - Network Layer
          • Responsible for communication between inter- and intra-networks
            • Intra-network: i.e. like LAN, same network
            • Inter-network: facilitates communication between major groups of networks
        • 2 - Datalink Layer
          • Facilitates communication between two devices on the same network
        • 1 - Physical Layer
          • Transferring bits of data into a physical cable
      • General model for network communication between two systems
